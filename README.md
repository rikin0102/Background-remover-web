# Background-remover-web
This is a simple Flask-based web application that removes the background of uploaded images using the Rembg library.
Users can upload any image (PNG, JPG, JPEG, WEBP), and it automatically generates a transparent background image.

#Features :
            -Upload image from browser
            -Removes background automatically
            -Shows original and processed image
            -User-friendly interface
            -Supports PNG, JPG, JPEG, WEBP
            -Files saved in static/uploads folder
            
#Technologies Used : Python, Flask, Rembg, Pillow(PIL), HTMLS/CSS
